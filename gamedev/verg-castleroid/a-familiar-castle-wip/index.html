Tile Pallet System

Work continues on my Castlevania 1 retro redux.  The Hall is coming along nicely.  Trying to focus on values, colors, and getting the tile system in place.  Much of the work isn’t shown in the simple graphic above, but I’m challenging myself to create a system that would theme of the environment to allow for a great deal more geometry.

Below is a look at a work in progress of the tile system.  I’ve decided make a theme that allows flaws in the execution of the system to be evident, and quickly fixable.  Skins would be painted over these prototype foundation assets.
