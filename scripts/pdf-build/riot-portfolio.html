Broadcast Production

Live broadcast overlay designed against in-game UI, sponsor lockup, and chat-readable transparency

03 / 09

Back2Warcraft Coverlay — broadcast overlay system for the long-running Warcraft III shoutcasting channel. Production-safe over live game footage, with chyron, sponsor, and channel-branding regions composited cleanly around the game's HUD.

  • Safe areas tuned against the in-game UI for chat-readable transparency.
  • PSD layer groups structured so production can re-key per event without designer involvement.
  • Identity refresh of the Back2Warcraft mark carried through the broadcast system.

Motion · Streaming Brand System

BaseTrade TV — Cheermote Family

Twitch-integrated animated cheermote family — pixel-art motion built as one cohesive brand

06 / 09

Animated cheermote family for the BaseTrade TV streaming ecosystem. Pose, palette, idle loop, and impact frame were designed once and applied across the family — each cheermote reads as the same brand at 112px chat scale. Native delivery as Twitch-compatible animated GIFs.
