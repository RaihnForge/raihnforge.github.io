Interactive SQL Visualizer — a Concept

August 9, 2025

A concept for transforming SQL queries into visual events in a top-down RPG-style town. Every command becomes a building, villager, or piece of infrastructure.

This concept transforms SQL queries into visual events in a top-down RPG-style town visualizer. Every SQL command becomes a building, villager, or infrastructure change, creating a living, evolving data town.

Event Mapping

SQL ConceptVisualizer Action
SELECT ... FROMSpawn a house for each table touched
WHEREDraw roads to filtered houses
JOINBuild walls or bridges between table houses
GROUP BYPlace a market square with stalls
ORDER BYAlign road lights in sequence
INSERTAdd villagers entering town
UPDATEUpgrade a building level
DELETEDemolish a structure
CREATE TABLELay foundations
DROP TABLECollapse ruins with a dust puff

Sample Visualizer Output

The idea grew out of wanting to teach SQL to people who learn visually, and — honestly — out of wanting to see what a query does to a world instead of what it does to a result set.
