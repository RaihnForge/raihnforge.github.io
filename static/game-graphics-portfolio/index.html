Back2Warcraft — Live Coverlay System

Broadcast Production Design · 2026 · Warcraft III: Reforged

Designed and delivered the on-broadcast overlay system for Back2Warcraft, the long-running shoutcasting channel covering Warcraft III World Championship Series and international events. The "Coverlay" is engineered to sit cleanly over live in-game footage — chyron, sponsor, and channel-branding regions composited around the game's HUD without obscuring competitive information.

  • Production-safe over live game footage — safe areas tuned against in-game UI overlap.
  • PSD layer groups structured so the production team can re-key per event without designer involvement.
  • Identity refresh of the Back2Warcraft mark carried through the broadcast system.

Approach

How I think about production design and visual systems work.

Production-Aware

Design for the live source

Overlay systems are tuned against the actual in-game UI — readable at 1080p over chaotic frames, safe against HUD overlap, and structured so the production team can re-key per event.

Systems Thinking

Marks and assets that scale

Identities ship as systems — primary mark, mascot variant, watermark size, print lockup. The same brand holds from icon to billboard, from desktop overlay to vertical social cut.

UX Lineage

Information hierarchy first

Whether it's a pick-phase overlay, a game UI screen, or a cheermote tier, the work starts with the user's read order. Visual hierarchy then carries the brand.
