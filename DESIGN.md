# Portfolio Website (RaihnForge) — Design Vision

> **Document Created:** 2026-04-03
> **Status:** Initial

## Vision

Twenty years of creative work, presented with the clarity and restraint that positions a Product Owner. The portfolio communicates systems-level thinking about product and design to executive audiences. No decorative noise — just the work, organized and accessible.

## Design Principles

1. **Content-first** — The work speaks. Layout serves content, never competes with it. 255+ pieces across visual design, game development, and journal entries.
2. **Fast and static** — Hugo builds 1463 pages in 400ms. No JavaScript in the reader-facing site. GitHub Pages hosting.
3. **Accessible editing** — Decap CMS editorial workflow allows content updates without git knowledge.
4. **Dual theme** — Light mode primary (professional, portfolio-appropriate), dark mode opt-in via `[data-theme="dark"]`.

## UX Philosophy

Three content sections:
- **Visual Design** — Design work, creative process, systems thinking
- **Products** — Game development projects, shipped work
- **Journal** — Reflections, process notes, industry observations

Recovered content (from WordPress migration) marked with muted notice. Clean navigation, fast page loads, no client-side rendering.

## Aesthetic Direction

- **Light mode primary:** Off-white background (#FAFBFE), burnt orange accent (#D4722A)
- **Typography:** Inter for body (modern, clean), Source Serif 4 for display (editorial, sophisticated)
- **Layout:** Clean sections, no decorative flourishes
- **Professional tone** — This is a portfolio for executive audiences, not a dev blog

## What Success Feels Like

A hiring manager lands on the site, sees twenty years of creative leadership organized into clear categories. They click into a project and understand the thinking behind it — not just the output. The site loads instantly. Dark mode is available for the dev-minded visitor. Content updates happen through Decap CMS without touching git.
